# Fonts Directory

Place the following font files in this directory:

1. **Panchang-Variable.ttf** - Used for headers (H1, H2, and bold accents)
2. **Manrope-VariableFont_wght.ttf** - Used for all body text

The fonts are configured in `app/fonts.ts` using Next.js's `next/font/local` for optimal performance.
